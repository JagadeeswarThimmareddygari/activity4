HTML - Hyper Text Markup Language

CSS - Cascading Style sheet

JavaScript - is a programming language that conforms to the ECMAScript specification

HTML stands for "HYPER TEXT MARKUP LANGUAGE".It's a markup language(uses tags to define element within document) ,not programmable language.HTML is the basic structure element used to create webpages.

1997-First standard browser started to pay attention.which is HTML 4.0 & 4.01.
2000-W3C came up with another specification called XHTML 1.0 and that specification was based on XML(very high rigid but very clear markup language) .
2004-WHATWG HTML(web hypertext application technology group)It's a bunch of browser vendors that got together.They have one central and one editor that makes the final decisions. WHATWG and W3C start to work together and produced in the end is HTML 5.19


W3C---charge of standard HTML5 (standard)
WHATWG--HTML (evolving)
caniuse.com --link to know the standards of HTML5,SVG,CSS,JS APIs..
validator.w3.org --we can upload html file or you can paste the html to check it's valid or not.
